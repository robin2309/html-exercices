# Exercices HTML & JavaScript

Bienvenue ! Ce dépôt contient des exercices pour apprendre les bases du HTML et du JavaScript.

## Prérequis

Aucune installation nécessaire. Tu as juste besoin d'un **navigateur web** (Chrome, Firefox, Safari…).

## Comment faire un exercice

1. Ouvre le dossier de l'exercice choisi
2. Double-clique sur le fichier `sujet.html` pour l'ouvrir dans ton navigateur
3. Ouvre également `sujet.html` dans un éditeur de texte (Notepad, VS Code…) pour modifier le code
4. Une fois que tu as terminé, compare avec `correction.html` pour voir une solution possible

> **Conseil :** essaie toujours de faire l'exercice par toi-même avant de regarder la correction !

## Exercice 1 — Ma page personnelle

📁 `exercice-1-page-perso/`

**Ce que tu vas apprendre :**
- La structure de base d'une page HTML (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Les balises de texte : `<h1>`, `<p>`, `<ul>`, `<li>`
- Insérer une image avec `<img>`
- Réagir à un clic avec JavaScript (`onclick`, `getElementById`)

**Résumé :** Tu vas créer une page web qui te présente — ton prénom, une courte bio, tes loisirs — et ajouter un bouton qui affiche un message secret quand on clique dessus.

---

## Exercice 2 — Jeu de devinette

📁 `exercice-2-jeu-devinette/`

**Ce que tu vas apprendre :**
- Récupérer la saisie d'un utilisateur avec `<input>`
- Lire la valeur d'un champ en JavaScript (`.value`, `parseInt`)
- Écrire une logique conditionnelle (`if / else if / else`)
- Afficher un résultat dans la page (`innerHTML`)

**Résumé :** Tu vas programmer un petit jeu où l'ordinateur choisit un nombre secret et le joueur doit le deviner. Le jeu indique si la proposition est trop grande, trop petite ou correcte.
